# 📘 TypeScript Studies

A comprehensive roadmap of my journey learning **TypeScript** and its integration with the **React** ecosystem. This repository serves as a personal documentation of concepts, syntax, and hands-on project implementation.

## 🗂️ Repository Structure

The repository is organized into three main stages of learning:

### 1. 🟢 Basics (`/basics`)
Fundamental TypeScript concepts independent of frameworks. 
- **Key Concepts**: Basic types (string, number, boolean), Interfaces vs. Types, Enums, Unions, and Functions.
- **Goal**: Understanding the compiler and the safety TypeScript adds to standard JavaScript.

### 2. ⚛️ React Basics (`/react-basics`)
Transitioning into the Frontend world using React with strict typing.
- **Key Concepts**: Typing Hooks (`useState`, `useEffect`), handling `SyntheticEvent`, Component Props, and `Children` types.
- **Goal**: Mastering how React and TypeScript communicate to prevent runtime errors.

### 3. 🚀 Projects (`/projects`)
Real-world applications where the previous concepts are applied.
- **Featured Projects**:
    - **Crypto Search**: API integration with CoinCap, dynamic routing, and professional financial formatting.
    - **devFrases**: State management with categories and random generators.
    - **Fuel Comparison**: Business logic implementation and string manipulation.

## 🛠️ Tech Stack & Tools
- **Language**: TypeScript 5.x+
- **Frontend**: React 19 (Hooks, Router Dom v7)
- **Icons**: React Icons
- **Build Tool**: Vite
- **Styling**: CSS Modules

## ⚙️ How to Explore
Each folder in the `/projects` directory is a standalone Vite project. To run any of them:
1. Navigate to the project folder: `cd projects/project-name`
2. Install dependencies: `npm install`
3. Start the dev server: `npm run dev`

## 📈 Learning Objectives
*   Mastering the **Strict Mode** in TypeScript.

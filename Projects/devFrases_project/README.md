# 💡 devFrases

A simple and interactive React application that generates inspirational phrases based on selected categories. Built with **React** and **TypeScript**.

## 📋 About the Project
**devFrases** is a project designed to practice React state management and conditional rendering. It allows users to toggle between different categories (Motivation, Good Morning, Good Night) and generate a random quote from that specific group.

## 🛠️ Tech Stack
*   **React 19** (`useState`)
*   **TypeScript** (Interfaces for data modeling)
*   **CSS Modules / Plain CSS** (Scoped styling)
*   **Vite** (Development environment)

## ⚙️ Key Features
- [x] **Category Selection**: Switch between different quote themes with a visual indicator for the active category.
- [x] **Random Generator**: Logic to pick a random index from the selected category's array.
- [x] **Conditional Rendering**: The phrase display area only appears once a quote has been generated.
- [x] **Template Strings**: Phrases are dynamically formatted with quotation marks upon generation.
